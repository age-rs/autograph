# Neural Network MNIST
This example shows how to train a Neural Network on the MNIST dataset.

# Usage
```
// Linear model
cargo run --release -- --linear
// Convolution Neural Network
cargo run --release -- --cnn
// The LeNet5 Network
cargo run --release -- --lenet5
```
